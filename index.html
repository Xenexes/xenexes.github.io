About

I’m a full stack developer from Germany, always searching for the next challenge. Currently I’m working for a German company which provides software solutions for route planning, fleet-, workshop-, depot- and infrastructure management. My main focus is the design and development of data-centric apps, based on web technologies, which can basically run on any platform (iOS, Android or in any web browser). The main focus of these apps are the extraction and simplification of specific business workflow.

Design

My experience on the front-end development reaches from simple CSS & HTML5 to Sass or frameworks like Angular 2+, Ionic, Cordova and whatever!

Code

My main development languages are Java and JavaScript / Typescript, SQL in combination with frameworks and technologies like JavaEE, JPA / Hibernate, REST, SOAP, Node.js and Docker. But I have also worked with C#, Python, Spring and more.

Tools
